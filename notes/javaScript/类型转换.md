# 类型转换

ECMAScript 语言会根据需要隐式执行自动类型转换

## ToBoolean(argument)

 转换成布尔值
|   类型    | 结果                                              |
| :-------: | :------------------------------------------------ |
| Undefined | 返回false                                         |
|   Null    | 返回false                                         |
|  Number   | 如果argument是+0，-0，或NaN返回false;否则返回true |
|  String   | 如果argument是空字符串返回false;否则返回true      |
|  Symbol   | 返回true                                          |
|  BigInt   | 如果argument是0返回false；否则返回true            |
|  Object   | 返回true                                          |
